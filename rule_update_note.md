# Terran: Territory Conquest - 규칙 개정 노트

## 버전 13 (v13) 변경 사항

- **개정일**: 2025년 7월 31일
- **주요 변경 내용**: 점령 인구 고정, 수도 겹침 방지, 인구/군사 관리 규칙 추가, 세력 성향 재부여 주기 도입.

| 항목 | 이전 규칙 (v12) | 개정된 규칙 (v13) |
|---|---|---|
| **점령 비용** | - 인구 100으로 시작, 100턴마다 20명씩 증가 | - **인구 100으로 고정** |
| **수도 지역** | - 항상 9칸 확보, 지형/호수 영향 없음 | - **다른 세력과 겹칠 수 없음** |
| **인구 관리** | - 영토 비례 생산 | - **3만 초과 시 빈 땅에 우선 소모** |
| **군사 관리** | - 총 군사력 1,000명 제한 | - **1,000명 도달 시 생산 중단, 900명 이하 시 재생산** |
| **AI 특성** | - 500턴 이후 부여 | - **300턴마다 세력 성향 재부여 가능** |

### 개정 의도

- **게임 플레이 예측 가능성 및 전략성 강화**: 점령 비용을 고정하고 수도 겹침을 방지하여 플레이어가 더 명확한 전략을 수립할 수 있도록 합니다.
- **자원 관리의 중요성 증대**: 인구 및 군사 관리 규칙을 추가하여 자원 활용에 대한 전략적 깊이를 더합니다.
- **AI의 동적 변화**: 세력 성향 재부여를 통해 게임 후반부에도 AI의 행동에 변화를 주어 게임의 재미를 유지합니다.

## 버전 11 (v11) 변경 사항

- **개정일**: 2025년 7월 31일
- **주요 변경 내용**: 지형 점령 턴 감소, 수도 지역 확보 규칙 강화, 산악 방어 보너스 추가, 점령 비용 동적 변화, 수도 안전 시 호전성 증가, AI 행동 로직 세분화.

| 항목 | 이전 규칙 (v10) | 개정된 규칙 (v11) |
|---|---|---|
| **지형 점령 턴** | - 평지 2턴, 농지 3턴, 산 1턴 | - **평지 1턴, 농지 2턴, 산 1턴** (1턴씩 감소) |
| **수도 지역** | - 주변 8칸(총 9칸)으로 구성, 호수에 잘릴 수 있음 | - **항상 9칸 확보**: 지형이나 호수에 잘리지 않고 9칸을 확보 |
| **산악 전투** | - 방어 보너스 없음 | - **방어력 15% 증가**: 산에서 방어하는 쪽은 방어력 15% 증가 |
| **점령 비용** | - 인구 100 고정 소모 | - **동적 비용**: 인구 100으로 시작, 100턴마다 50%씩 증가<br>- 현재 점령 비용 정보 패널에 표시 |
| **호전성** | - 고정된 호전성 수치 | - **수도 안전 시 증가**: 수도 중심부 5칸 이내에 적이 없으면 호전성 20% 상승 |
| **AI 행동 로직** | - 초반 확장, 중반 이후 수도 방어 및 복수심 | - **500턴까지 확장 우선**: 500턴 이후 세력별 특성(확장주의자, 정복자, 방어자)에 따라 행동 우선순위 부여 |

### 개정 의도

- **전략적 깊이 증대**: 지형 점령의 부담을 줄이고, 산악 지형의 전략적 가치를 높여 다양한 전술을 유도합니다.
- **게임 후반부 동적 변화**: 점령 비용의 증가와 AI 특성 도입으로 게임이 진행될수록 새로운 도전과 예측 불가능성을 제공합니다.
- **AI의 다양성**: 각 AI 세력이 고유한 전략적 성향을 가지도록 하여 게임 플레이의 재미를 더합니다.

## 버전 10 (v10) 변경 사항

- **개정일**: 2025년 7월 31일
- **주요 변경 내용**: 수도 함락 및 세력 흡수 규칙을 도입하여 게임의 목표를 더 명확히 하고, AI에 복수심 시스템을 추가하여 게임 후반부의 변수를 극대화했습니다.

| 항목 | 이전 규칙 (v9) | 개정된 규칙 (v10) |
|---|---|---|
| **게임 턴** | - 최대 1000턴 | - **최대 2500턴**으로 증가 |
| **승패 조건** | - 최후 생존 또는 턴 종료 시 영토 1위 | - **수도 함락 및 세력 흡수 추가**:<br>  - 세력의 수도 중심부가 점령되면 해당 세력은 즉시 멸망<br>  - 멸망한 세력의 모든 자산(영토, 군인)은 승리자에게 흡수됨 |
| **AI 전략** | - 301턴부터 수도 방어 | - **복수심(Vendetta) 시스템 추가**:<br>  - 수도권(9칸)이 공격받으면, 해당 적에게 총공격 감행 |
| **UI** | - 호전성 수치 미표시 | - **호전성 수치 표시**: 정보 패널에 호전성 수치를<br>  표시하도록 코드를 검토하고 수정 |

### 개정 의도

- **게임 목표의 명확화**: '최후의 1인'이라는 목표를 '수도 함락'이라는 구체적인 행동과 연결하여, 플레이어(AI)가 무엇을 해야 할지 더 명확하게 인지하도록 했습니다. 세력 흡수 규칙은 큰 위험을 감수한 공격에 대한 확실한 보상을 제공합니다.
- **감정적인 AI 구현**: AI가 단순히 계산에 따라 움직이는 것을 넘어, 자신의 핵심 영토를 침범한 적에게 '복수'하는 감정적인 행동 패턴을 추가했습니다. 이는 게임 후반부의 예측 불가능성을 높이고, 플레이어가 AI의 행동을 유발하는 전략을 사용할 수도 있게 합니다.
- **장기적인 플레이 지원**: 최대 턴을 늘려, 다양한 전략과 역전의 기회를 충분히 탐색할 수 있는 시간을 제공했습니다.

## 버전 4 (v4) 변경 사항

- **개정일**: 2025년 7월 31일
- **주요 변경 내용**: AI의 확장 로직을 개선하고, 초반 군사력 증강 규칙을 추가했으며, 전투 공식을 수정하여 게임의 역동성을 강화했습니다. 또한, 가시성 개선 및 로그 기능을 추가했습니다.

| 항목 | 이전 규칙 (v3) | 개정된 규칙 (v4) |
|---|---|---|
| **확장 AI** | - 진격 방향 우선, 이후 무작위 탐색 | - **빈 땅 우선 점령**: 주변의 빈 땅을 최우선으로 공격<br>- 빈 땅이 없을 때만 다른 세력 공격 |
| **군인 생산** | - 호전성에만 의존하여 생산 | - **초반 군사력 증강**: 총 군인 100명까지 호전성 무시하고<br>  인구를 최대한 군인 생산에 투자 |
| **공격 부대 규모**| - 셀 군인의 50%를 투입 | - **소규모 그룹 공격**: 5~10명의 군인을 무작위로 차출하여 공격 |
| **전투 공식** | - 선형적 확률: `공격력 / (공격력 + 방어력)` | - **제곱 비례 확률**: `공격력^2 / (공격력^2 + 방어력^2)`<br>- 병력 수 차이가 승패에 더 큰 영향을 줌 |

## 버전 2 (v2) 변경 사항

- **개정일**: 2025년 7월 31일
- **주요 변경 내용**: 게임의 전략성 강화를 위해 인구, 군인, 전투 시스템을 전면 개편하고, 세력별 진격 방향 개념을 도입했습니다.

| 항목 | 초기 규칙 (v1) | 개정된 규칙 (v2) |
|---|---|---|
| **진격/확장** | - 방사형 확장: 군인이 있는 셀 주변 4방향으로 무작위 확장 | - **지정 방향 진격**: 각 세력은 고유의 진격 방향을 가짐<br>- 해당 방향을 우선적으로 공격하고, 없을 시 주변 8방향 탐색 |
| **인구 시스템** | - 셀 기반 인구: 각 셀이 개별적으로 인구를 가짐<br>- 턴당 1씩 자연 증가 | - **세력 통합 인구**: 인구가 세력 단위로 통합 관리됨<br>- **영토 비례 생산**: 영토(셀) 1개당 턴마다 3~10 인구 생산 |

| **전투 시스템** | - 결정론적 전투: 공격 측 군인 > 방어 측 군인이면 무조건 승리 | - **확률적 전투**: 양측 군인 수에 비례한 확률로 승패 결정<br>- (예: 공격 10 vs 방어 5 -> 공격 측 승리 확률 66.6%) |
| **UI** | - 턴 정보만 표시 | - **정보 패널 추가**: 화면 우측에 패널을 추가하여<br>  세력별 총 인구, 총 군인, 영토 수를 실시간으로 표시 |
| **초기 설정** | - 시작 시 각 세력은 인구 10을 가짐 | - 시작 시 각 세력은 군인 5명과 인구 150을 가짐 |
